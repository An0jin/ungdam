# 웅담 초등학교 방과후 컴퓨터 수업 자료

웅담초등학교 초등학교 3~4학년 및 5~6학년 학생들을 위한 방과후 수업 저장소입니다.

## 🛠 학년별 교육 과정 및 도구

본 과정은 학년이 올라감에 따라 기본 도구에서 AI 기반 도구로 자연스럽게 전환하는 **도구 업그레이드** 전략을 따릅니다.

| 학년              | 구분           | 주요 도구                              | 학습 목표                                 |
| :---------------- | :------------- | :------------------------------------- | :---------------------------------------- |
| **3~4학년** | **기초** | Chrome, 메모장                         | 인터넷 검색 및 텍스트 편집 기본기         |
| **5~6학년** | **심화** | **FlowithOS**, **VS Code** | AI 웹 브라우저 및**AI 메모장** 활용 |

---

## 🎯 수업의 핵심 방향

본 수업은 단순히 컴퓨터 기능을 익히는 것을 넘어, **"도구의 진화"**를 체감하는 데 중점을 둡니다.

---

## 🏫 [3~4학년반] "내 손으로 만드는 디지털 기초"

**도구**: Chrome 브라우저, 메모장

* **수업의 느낌**: "컴퓨터라는 신기한 도구와 친해지기"
* **어떻게 배우나?**:
  * **Chrome**: 넓은 인터넷 세상에서 내가 원하는 정보를 **정확하게 찾아내는 법**을 연습합니다. 낚시성 정보나 광고를 구별하는 '디지털 시력'을 기릅니다.
  * **메모장**: 화려한 꾸미기 없이 오직 **내 생각과 정보**를 텍스트로 옮기는 기초를 다집니다. 파일 저장과 관리라는 컴퓨터의 가장 기본 원리를 몸으로 익힙니다.
* **목표**: "스스로 정보를 찾고, 그것을 내 글로 정리할 수 있다."

---

## 🚀 [5~6학년반] "AI와 함께하는 미래형 작업"

**도구**: **FlowithOS** (Chrome 업그레이드), **VS Code** (메모장 업그레이드)

* **수업의 느낌**: "AI를 비서로 부리며 생산성 폭발시키기"
* **어떻게 배우나?**:
  * **FlowithOS**: 더 이상 사람이 일일이 검색하지 않습니다. AI 웹 브라우저를 통해 **AI 에이전트에게 목적을 전달**하고, AI가 웹을 누비며 가져온 결과를 검토하고 조립합니다.
  * **VS Code**: 단순한 메모를 넘어 **구조화된 정보**를 다루는 **"AI 메모장"**으로 활용합니다. AI가 정리해준 방대한 정보를 체계적으로 관리하고, AI와 협업(Co-creation)하여 나만의 콘텐츠를 완성하는 과정을 체험합니다.
* **목표**: "AI 도구를 활용해 복합적인 문제를 해결하고 나만의 디지털 결과물을 만든다."
